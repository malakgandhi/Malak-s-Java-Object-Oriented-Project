## Description
A Java-based application using OOP concepts and JDBC to manage student records.

## Features
- Add Student
- View Students
- Update Student
- Delete Student
- Search Student

## Technologies Used
- Java
- MySQL
- JDBC

## How to Run
1. Import project into Eclipse
2. Add MySQL Connector JAR
3. Run student_db.sql in MySQL Workbench
4. Update DB credentials in DBConnection.java
5. Run Main.java
